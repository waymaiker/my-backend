## Installation

```bash
$ npm install
```

## Running the app

```bash
# Seeding the database
$ bash ./seed_script_db.sh seed

# Reset the databse
$ bash ./seed_script_db.sh reset

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
